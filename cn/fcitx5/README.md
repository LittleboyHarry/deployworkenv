# fcitx5 调优

## `cn/fcitx5/coder`

程序员编程专用优化，解除快捷键冲突：

- 以 `Super + Space` 代替 `Ctrl + Space`
- 解除 `Ctrl + Shift + F` 简繁体切换键

## `cn/fcitx5/recommend`

- 不因应用而切换语言
- `Super + V` 弹出剪贴板
- 候选框微调：竖排大号字体外观

## `cn/fcitx5/gnome`

在 GNOME 中必用，功能：

- 安装 [kimpanel](https://extensions.gnome.org/extension/261/kimpanel/)
  扩展改善外观：大号字体竖排候选框
